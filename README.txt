ATPG.py implements the PODEM algorithm and a logic simulator in python.
Command to run from terminal:
>python ATPG.py

Version -- Python 3.7.0
Libraries imported -- copy

Input format: 
	Input nets:
	Output nets:
	Fault: Faulty Net, Stuck at (1/0)
	Netlist

Output format: 
	Test Vector (logic value for each primary input)

Test vector also printed to stdout.

Change the filepath in ATPG.py line no. 23 and 438 to change input/output txt files.
